# AI Job Market Dashboard (2025–2026)

This project explores trends in the global AI job market using Python and Streamlit.

The dashboard analyzes:

• Salary distribution  
• Jobs by experience level  
• Top countries hiring AI professionals  
• Most requested AI skills  

Tools used:

Python  
Pandas  
Seaborn  
Streamlit  

Dataset: AI Jobs Market 2025–2026
